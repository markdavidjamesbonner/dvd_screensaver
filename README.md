# DVD Screensaver

### Description
A nostalgic CSS/HTML recreation of the classic DVD screensaver logo bounce effect.

### How It Works
This project uses vanilla HTML, CSS, and JavaScript to simulate the iconic DVD logo's movement and bounce against the edges of the screen.

### Tech Stack
- HTML
- CSS
- JavaScript

### Key Features
- Responsive design.
- Smooth animation effects resembling the original screensaver.
